# Security Policy

This project is a personal productivity tool. It may handle sensitive note content locally (as part of export/backup). Please review the code and use at your own risk.

## Reporting a vulnerability

Please do **not** open a public GitHub issue for security-related reports.

Instead, email:

- `javi@javisoto.es`

Include:

- A description of the issue and impact
- Reproduction steps (or a minimal PoC)
- Affected versions / commit hash
